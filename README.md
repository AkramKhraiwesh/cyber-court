# cyber-court


## Overview
Cyber Court is an AI-based cybersecurity legal advisor system developed as a graduation project. The system is designed to support cybercrime investigations by combining cybersecurity analysis, digital evidence management, and legal case analysis in one platform.

The project mainly focuses on phishing attacks and financial fraud investigations through an interactive and user-friendly environment.

---

## Features
- Cybercrime case management
- Digital evidence organization
- AI-assisted case analysis
- Legal advisory and law mapping
- Interactive timelines and dashboards
- Investigator and suspect profiles
- Case visualization and summaries

---

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
- SQL Database
- NLP (Natural Language Processing)

---

## System Components

### Case Management Module
Manage investigation cases, case details, and case status.

### Evidence Management Module
Upload and organize digital evidence such as emails, URLs, and transaction records.

### AI Analysis Module
Analyze case descriptions and extract important entities and events.

### Legal Advisory Module
Map cybersecurity findings to related cybercrime laws and legal concepts.

### Visualization Dashboard
Display timelines, dashboards, and investigation summaries.
